# Contributing to WeavScope

First off, thank you for considering contributing to WeavScope! It's people like you that make WeavScope such a great tool.

### 1. Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](https://github.com/mmycin/inframan/issues) to see if it has already been reported. If not, feel free to open a new one!

### 2. Fork & create a branch

If this is something you think you can fix, then fork the repository at `github.com/mmycin/inframan` and create a branch with a descriptive name.

### 3. Get the test suite running

Make sure you install the development dependencies and ensure the tests pass:

```bash
uv sync
uv run pytest
```

### 4. Implement your fix or feature

At this point, you're ready to make your changes. Feel free to ask for help; everyone is a beginner at first.

### 5. Make a Pull Request

At this point, you should switch back to your master branch and make sure it's up to date with WeavScope's master branch.
Then push your feature branch and open a Pull Request!
